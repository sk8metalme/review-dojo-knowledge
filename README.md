 # PR Review Knowledge Base

  このリポジトリは、PRレビューから抽出された有益な知見を蓄積しています。

  ## 構成

  - `security/` - セキュリティ関連の知見
  - `performance/` - パフォーマンス関連の知見
  - `readability/` - 可読性・命名関連の知見
  - `design/` - 設計・アーキテクチャ関連の知見
  - `testing/` - テスト関連の知見
  - `error-handling/` - エラーハンドリング関連の知見
  - `other/` - その他の知見
  - `archive/` - アーカイブされた古い知見

  ## 自動収集

  このリポジトリは GitHub Actions により自動的に更新されます。
